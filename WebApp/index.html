      Acerca de la aplicación

      Esta aplicación es creada para visualizar de una manera rápida, sin tantas complicaciones el pico y placa en las principales ciudades de Colombia. Aplicación creada con estándares de la W3 con HTML5 y compilada con Intel XDK, Website creado con el template de Start Bootstrap
